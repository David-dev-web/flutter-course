# Flutter Offline Masterclass

Dies ist eine moderne, offline-fähige Lernplattform für Flutter-Anfänger.

## Features
- **Nothing OS Design**: Minimalistisch, dunkles UI mit roten Akzenten.
- **Offline First**: Alle Inhalte, Quizze und Beispiele funktionieren ohne Internet.
- **Fortschritts-Tracking**: Speichert deinen Lernfortschritt lokal im Browser.
- **Interaktives Terminal**: Simuliert eine Flutter-Konsole für bessere Motivation.
- **Achievement System**: Sammle Erfolge für abgeschlossene Kapitel.

## Installation
1. Lade den Ordner `flutter_offline_course` herunter.
2. Öffne die `index.html` in einem beliebigen modernen Browser.

## Struktur
- `assets/css`: Styling und Animationen.
- `assets/js`: Logik, Daten und Inhalte.
- `index.html`: Der Einstiegspunkt.
